# ShaadiBio – Marriage Biodata Generator

This project is a web application that allows users to create a marriage biodata using customizable templates and download it as a PDF.

Features:
- User registration and login
- Personal details form with age calculation
- Family details section
- Education and profession section
- Optional horoscope section
- Profile photo upload
- Drag and drop photo upload
- Two biodata templates
- Live biodata preview
- Download biodata as PDF
- Save biodata using backend

Tech Stack:
Frontend:
React, TypeScript, TailwindCSS

Backend:
Node.js, Express, MongoDB
